# Computational Physics Laboratory (PH3120)

This repository contains my laboratory exercises, assignments, reports, and revision materials completed for the **Computational Physics (PH3120)** module during the **3rd Year** of my **B.Sc. (Hons) in Electronics and Information Technology**.

The work focuses on applying numerical methods and scientific computing techniques using **Python** and **Jupyter Notebook** to solve physics and engineering problems through computational approaches.

---

## 📚 Course Overview

This repository covers a wide range of computational physics topics, including:

- Python programming for scientific computing
- Numerical differentiation
- Numerical integration
- Root-finding algorithms
- Interpolation and curve fitting
- Solving ordinary differential equations
- Matrix operations and linear algebra
- Scientific data visualization
- Computational modelling
- Chaos and nonlinear systems
- Monte Carlo simulations

---

## 🛠 Tools and Libraries Used

- Python
- Jupyter Notebook
- NumPy
- SciPy
- Matplotlib
- Pandas


---

## 📅 Weekly Laboratory Exercises

### Week 01 – Introduction to Python & Scientific Computing (CPL100)
- Introduction to Python programming
- Jupyter Notebook environment
- NumPy arrays and mathematical operations
- Basic plotting using Matplotlib

### Week 02 – Numerical Integration (CPL101)
- Trapezoidal Rule
- Simpson's Rule
- Numerical approximation of definite integrals
- Error comparison

### Week 03 – Ordinary Differential Equations (CPL102)
- Euler Method
- Improved Euler Method
- Numerical solutions of first-order ODEs

### Week 04 – Higher-Order Differential Equations (CPL103)
- Initial value problems
- Numerical solution techniques
- Accuracy analysis

### Week 05 – Numerical Linear Algebra (CPL104)
- Matrix operations
- Gaussian Elimination
- Solving systems of linear equations

### Week 06 – Interpolation & Curve Fitting (CPL105)
- Polynomial interpolation
- Least Squares Method
- Curve fitting
- Data visualization

### Week 07 – Numerical Differentiation (CPL106)
- Forward Difference
- Backward Difference
- Central Difference
- Error analysis

### Week 08 – Root Finding Methods (CPL107)
- Bisection Method
- Newton–Raphson Method
- Secant Method
- Convergence analysis

### Week 09 – Computational Modelling (CPL108)
- Mathematical modelling
- Scientific simulations
- Visualization of computational results

### Week 10 – Chaos in Continuous Systems (CPL109)
- Lorenz System
- Phase space analysis
- Chaotic behaviour
- Numerical simulations

### Week 11 – Monte Carlo Methods (CPL110)
- Random number generation
- Monte Carlo integration
- Statistical simulations

### Week 12 – Advanced Computational Physics (CPL111)
- Advanced numerical techniques
- Scientific computing applications

### Week 13 – Advanced Numerical Simulations (CPL112)
- Physics simulations
- Algorithm optimization
- Numerical experiments

### Week 14 – Final Laboratory Exercises (CPL113)
- Integration of numerical methods
- Complete computational physics problems
- Result analysis and report preparation
